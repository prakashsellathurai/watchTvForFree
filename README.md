# watchTV forfree

A simple, static web application to browse and stream IPTV channels directly in your browser. Powered by [iptv-org](https://github.com/iptv-org/iptv).

## Features
- **Region & Category Filtering**: Easily find channels by country or genre.
- **Live Search**: Instantly search for channels by name.
- **In-Browser Streaming**: Integrated HLS player for seamless playback.
- **Responsive Design**: Works on desktop, tablet, and mobile.
- **Dark Mode**: Premium, eye-friendly interface.

## How to Run Locally
1. Clone the repository.
2. Open `index.html` in your browser.
   - *Note: For best results, run a local server (e.g., `python -m http.server`) to avoid CORS issues.*
